牛蒡茶

首选来自“中国牛蒡之乡”—苍山

最后看冲泡出来的效果，一般好的牛蒡茶开水冲泡后有一股浓郁的人参香气，茶色红润，味道甘醇，耐冲泡。反之，一般的牛蒡茶就相差甚远。好的牛蒡茶见效，经常饮用会有口干、腹胀、放屁、排便功能增强、肚饿等类似的好转反应。而普通的则要差很多，而且很多达不到卫生、质量标准的牛蒡茶还会导致其他疾病。

如何选购牛蒡茶
⒈原料：原料直接决定了牛蒡茶的品质，选用来自“牛蒡之乡”苍山的牛蒡是最好的，因为那里有特别适合牛蒡生长的土壤以及新鲜干净的空气、水分，所以培育出的牛蒡营养丰富、生命力强，是中国牛蒡的源产地和出口首选地；
⒉外观：看茶片是否是非空心（茶片中心呈白色空洞状）（空心的牛蒡片一般是低端的），看是否为小圆片（小圆片是优质黄金牛蒡），茶水色泽是否呈红润的酒红色且无杂质；闻气味是否香浓。好的牛蒡茶开水冲泡后会有一股浓郁的人参或者咖啡的香味，且茶色红润；
⒊口感：好的牛蒡茶味道甘醇柔滑，耐冲泡，饮后口有余香、淡淡回甘。反之，一般的牛蒡茶就相差甚远；
⒋包装：包装不仅仅是体现牛蒡茶的档次，更是质量、效果的保证。看外包装是否有食品安全认证QS标志；看内包装是否严密，内包装最好采用铝膜纸，其保鲜和防水性极好。

注意事项
编辑
饮用时应该注意的事项，牛蒡茶在饮用时不宜过浓，过浓容易引起上火，就像我们平时喝高汤一样，喝的量多，就容易上火，在泡制的时候可以适当的加入一些蜂蜜这样就不会上火了！
还有牛蒡茶就有排毒养颜，减肥的功效，女性在饮用时要注意月经期最好不要饮用！
每天5-8克，便秘者、减肥者可酌情增加饮用牛蒡茶。
牛蒡茶不能冷饮，冷饮会导致腹泻。

喝牛蒡茶却有些注意事项。
⒈初饮者可能出现排便功能增强（不是水泻）现象。
2、因牛蒡茶有助于辅助稳定降低高血压，建议低血压患者间歇饮用。
3、不与其它茶混饮，泡发的茶片最后食掉最佳。
4、坚持饮用一个月就会有明显效果。

牛蒡茶色泽金黄、香味宜人，是老幼四季皆宜的保健饮品，也可炖、煮、涮、做汤等。[1] 牛蒡茶具有通经络活血化淤的作用早在李时珍的《本草纲目》中就有记载牛蒡能“通十二经脉，除五脏恶气”，能调整女性经期，对于女性的气滞血淤的症状和久坐办公室的上班族有很好作用。.牛蒡茶不宜过浓，过浓容易引起上火，在泡 制的时候可以适当的加入一些蜂蜜这样就不会上火了![3

具体做法：
热饮：每次5-8克（备注7到8片），便秘者、减肥者可酌情增加。
用开水冲泡2-3分钟即可饮用（可反复冲泡多次），也可以配以冰糖、红枣、洋参冲泡效果更佳，还可以放少许蜂蜜，预防上火！

营养价值：

牛蒡含菊糖、纤维素、蛋白质、钙、磷、铁等人体所需的多种维生素及矿物质，其中胡萝卜素含量比胡萝卜高150倍，蛋白质和钙的含量为根茎类之首。牛蒡根含有菊糖（人体内不含菊糖）及挥发油、牛蒡酸、多种多酚物质及醛类，并富含纤维素和氨基酸。
牛蒡根含有人体必需的各种氨基酸，且含量较高，尤其是具有特殊药理作用的氨基酸含量高，如具有健脑作用的天门冬氨酸占总氨基酸的25%-28%，精氨酸占18%-20%，且含有Ca、Mg、Fe、 Mn、 Zn等人体必需的宏量元素和微量元素。[6-7]

药用功效：
治疗糖尿病、高血压、高血脂、抗癌等作用。可降血糖、降血压、血脂、治疗失眠，提高人体免疫力等功效。
牛蒡全植物含有抗菌成分，其中叶含抗菌成分最多，主要抗金黄色葡萄球菌。
抗衰老和清除氧自由基作用
人类生命在正常活动代谢过程中，会产生一种有害于身体健康，促使细胞衰老的物质——氧自由基。它们能够促使产生脂褐斑色素——老年斑的生成和堆积。老年斑在体表的出现，表示机体中细胞已进入衰老阶段。牛旁根中含有过氧化物酶，它能增强细胞免疫机制的活力，清除体内氧自由基，阻止脂褐质色素在体内的生成和堆积，抗衰防老，为机体提供了对抗和清除氧自由基的内护环境。
抗癌作用
牛蒡苦素能抑制癌细胞中磷酸果糖基酶的活性，牛蒡甙元也有抗癌活性，同时牛蒡子甙元还具有抗老年性痴呆作用。现代科学研究结果认为，牛蒡含黄酮甙类化合物，对恶性肿瘤具有一定抗性，其粗提取物呈选择毒性，较低量就可以抑制癌细胞增殖，使肿瘤细胞向正常细胞接近。[9]
移除废水中重金属作用
牛蒡对金属离子的吸附能力依次为：Pb>Cd>Hg>Ca>Zn。
其它作用
牛蒡甙和牛蒡酚有抗肾炎活性
，能有效地治疗急性进行性肾炎和慢性肾小球肾，做肾病治疗剂。牛蒡的主要成分木酚素具有抑制血小板活化因子对血小板结合作用，可以做血小板活化因子拮抗药。[10]

####Unsupported compiler
Unsupported compiler ‘com.apple.compilers.llvmgcc42’ selected for architecture ‘armv7’

Xcode更新到Xcode5，一运行报如下错误：

Unsupported compiler ‘com.apple.compilers.llvmgcc42’ selected for architecture ‘armv7’或’armv7s’

解决方案：

Change your compiler for C/C++/ObjectiveC Go to Build Settings->Build OPtions->compiler for C/C++/ObjectiveC; select Default(Apple LLVM5.0)

AFNetworking

test1
step1 打开工具 Network Link Conditioner
step2 并发发起200个GET请求
step3 在mac上，同时发起的请求为4个。因为网络环境差，这四个请求都在等待。当四个请求的response回来后，又发起了4个请求。由此可以判断，在模拟机上，并行支持的get请求数为4.真机上待验证。
step4 真机同模拟器一样，也是4个请求一组的发送

结论：真机在wifi情况下，同时能处理的请求为4个NSURLConnection。
问题1:此次实验用的是AFNetworking，下次直接用NSURLConnection。
问题2:上一组请求完毕之后，要等一段时间才看到下一组请求，这是为什么？间隔期间做了什么？长达8s。是AFNetworking的漏洞吗？

appstore审核

＊需要增加“本app的活动与app Inc。无关的声明”
＊不能出现“检查更新”，因为appstore力推通过appstore来更新app
＊引导页要能表述app的基本内容
＊通过第三方APP登录注册的时候要先判断第三方是否已经安装。不能出现未安装而不能进入app的情况
＊17.2 注册要做成非必需的。
＊不能出现介绍其他app的字眼，或疑似信息
＊试用版账号的功能尽量少，如果试用版账号密码有修改，必需及时更新。
